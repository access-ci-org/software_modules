# software_modules
List software modules per resource or group


## Requirements
* docker


## Usage
* `curl
  https://raw.githubusercontent.com/access-ci-org/software_modules/refs/heads/main/go.sh | bash`
* `python report.py`


## Developer
* `git clone https://github.com/access-ci-org/software_modules.git`
* `cd software_modules`
* `make go`
